# Roy's Planck Keymap

## Keymap Visualization

keymap.yaml を下記サイトにアップロードしてSVGを生成できます。

https://keymap-drawer.streamlit.app/

## Layers

- **CUYZ**: メインレイヤー (オリジナルから `/` → `'` に変更)
- **LOWER**: 数字、Fキー、Vim操作
- **RAISE**: 記号、矢印キー
- **ADJUST**: マウス、Fキー (Lower + Raise)
- **NAGINATA**: 薙刀式入力
- **COMBO**: コンボ表示用

## Combos

| Keys | Output |
|------|--------|
| P + W | Enter |
| W + M | Shift+Enter |
| T + N | 薙刀式 ON/OFF |
| I + . | Save |
| E + I | Select All |
